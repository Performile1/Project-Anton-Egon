# Swedish Voice Recording Script
**Project Anton Egon - Voice Sample Recording**

This script contains phonetically balanced text covering most Swedish sounds, designed for high-quality voice recording for the Anton Egon agent.

---

## Recording Guidelines

### Technical Specifications
- **Sampling frequency:** Minimum 44.1 kHz (recommended 48 kHz)
- **Bit depth:** 24-bit
- **Format:** WAV (uncompressed)
- **Microphone:** Quality external microphone (not laptop built-in)
- **Environment:** Quiet room, no echo, "dry" sound

### Recording Tips
1. **Silence:** Include 2 seconds of silence at beginning and end of each file
2. **Consistency:** Match voice tone with video outfit (irritated voice = irritated outfit)
3. **Distance:** Maintain consistent microphone distance (15-20 cm)
4. **Posture:** Sit upright, consistent posture for all recordings
5. **Hydration:** Drink water before recording to avoid mouth clicks

## Recording Workflow

### Step 1: Setup
1. Connect microphone to computer
2. Set recording software to 48 kHz, 24-bit, WAV
3. Test microphone levels (aim for -12 dB to -6 dB)
4. Ensure quiet environment

### Step 2: Record Phonetic Balance
- Read Part 1 clearly and slowly
- Emphasize each vowel and consonant
- Repeat if necessary for clarity

### Step 3: Record Number Test
- Read Part 2 with precision
- Enunciate each number clearly
- Maintain consistent pace

### Step 4: Record Mood Test
- Read the same sentence 3 times with different emotions
- Professional/Neutral: Flat, clear, professional
- Irritated/Concise: Faster, sharper, slightly tense
- Happy/Collaborative: Warmer, more expressive, upward inflection

### Step 5: Record 50 Sentences
- Read each sentence clearly
- Maintain consistent distance from microphone
- Take short breaks between sentences
- Save each sentence as separate file or sections

### Step 6: Quality Check
- Listen to recordings for clarity
- Check for background noise
- Ensure consistent volume across recordings
- Re-record if necessary

## Tips for Best Results

1. **Hydration:** Drink water before recording to avoid mouth clicks
2. **Posture:** Sit upright with shoulders relaxed
3. **Breathing:** Take deep breaths before each sentence
4. **Pacing:** Speak at a natural, conversational pace
5. **Consistency:** Use the same microphone and setup for all recordings
6. **Environment:** Record in the same room with consistent acoustics
7. **Practice:** Read through the script once before recording
8. **Breaks:** Take short breaks to avoid vocal fatigue
