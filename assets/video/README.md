# Video Assets Structure

This directory contains pre-recorded video clips for the Wardrobe Engine.

## Required Video Assets

### Base Outfits (Idle Loops)
Each outfit needs a 30-60 second idle loop with consistent lighting and position.

- `outfit_shirt_01_idle.mp4` - Professional shirt outfit (idle loop)
- `outfit_shirt_02_idle.mp4` - Alternative shirt outfit (idle loop)
- `outfit_tshirt_idle.mp4` - Casual t-shirt outfit (idle loop)
- `outfit_glasses_idle.mp4` - Outfit with glasses (idle loop)
- `outfit_casual_idle.mp4` - Casual outfit (idle loop)

**Important Guidelines:**
- All videos must have identical lighting conditions
- Camera position must be consistent across all outfits
- Frame rate: 20-24 FPS (sufficient for Teams)
- Resolution: 1280x720 or 1920x1080
- Loop points: Ensure seamless looping (matching exit/exit frames)
- Background: Solid color or blurred office background

### Action Videos
Short action clips that transition from/to idle position.

- `action_drink_water.mp4` - Drinking water action (3-5 seconds)
- `action_check_phone.mp4` - Quick glance at phone (2-3 seconds)
- `action_adjust_glasses.mp4` - Adjusting glasses (1-2 seconds)
- `action_scratch_head.mp4` - Scratching head (2-3 seconds)
- `action_look_notes.mp4` - Looking at notes (3-4 seconds)
- `action_stretch.mp4` - Subtle stretch (2-3 seconds)
- `action_clear_throat.mp4` - Clearing throat (1 second)

**Action Video Guidelines:**
- Must start and end in the same position as idle loop
- Duration: 1-5 seconds depending on action
- Seamless transition points at start and end
- Same lighting and camera angle as idle loops

## Recording Tips

1. **Lighting Consistency**
   - Use the same lighting setup for all recordings
   - Avoid direct sunlight (changes throughout the day)
   - Use diffused lighting for soft, consistent shadows

2. **Camera Position**
   - Mark camera position with tape
   - Use tripod to prevent movement
   - Maintain same zoom level

3. **Loop Points**
   - Record longer clips (60-90 seconds) and edit to find best loop points
   - Use video editing software to match frames
   - Test loops before finalizing

4. **Action Videos**
   - Practice actions before recording
   - Ensure smooth transitions
   - Record multiple takes and choose best one

## File Naming Convention

- Base outfits: `outfit_[name]_idle.mp4`
- Actions: `action_[name].mp4`
- All lowercase with underscores

## Storage

- Estimated size per video: 10-50 MB (depending on length and quality)
- Total estimated storage: 200-500 MB for complete asset library
