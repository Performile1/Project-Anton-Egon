# Background Assets

This directory contains background images/videos for Anton Egon's video output.

## Background Types

### 1. Office/Professional Backgrounds
- `office_blur.jpg` - Blurred office background (professional)
- `office_clean.jpg` - Clean office background
- `office_window.jpg` - Office with window view

### 2. Home Office Backgrounds
- `home_office.jpg` - Home office setup
- `home_blur.jpg` - Blurred home environment
- `home_library.jpg` - Library background

### 3. Abstract/Modern Backgrounds
- `abstract_gradient.jpg` - Modern gradient
- `abstract_geometric.jpg` - Geometric pattern
- `abstract_clean.jpg` - Clean abstract

### 4. Brand/Corporate Backgrounds
- `corporate_logo.jpg` - Branded background with logo
- `corporate_clean.jpg` - Clean corporate background

## Guidelines

- Resolution: 1280x720 or 1920x1080
- Format: JPG or PNG (images), MP4 (video backgrounds)
- Color temperature: Neutral (avoid strong color casts)
- Lighting: Consistent with outfit lighting
- Blur: Slightly blurred backgrounds work best for video calls

## Usage

Backgrounds can be applied via:
- OBS Studio (background removal + replacement)
- Post-processing in video synthesis
- Compositing in wardrobe manager
